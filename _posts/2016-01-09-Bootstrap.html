If you're anything like me, I'm terrible at CSS and styling. Designing is not something I'm good at, I can barely wear clothes that color coordinate well together. Introduce CSS frameworks like Bootstrap and Foundation. However, as I learned the hard way, do NOT try to implement multiple CSS frameworks. I spent hours trying to figure out why certain things were not working, but it was due to having foundation css and bootstrap css both loaded at the same time. Here are a few reasons to use CSS frameworks. Speed - Bootstrap provides you with ready made blocks of code for any design related component you need. There are also bootstrap themes available to cut down time on producing any website. Responsiveness - One of the greatest features of frameworks like bootstrap is its ability to dynamically adjust to proper screen resolution.
